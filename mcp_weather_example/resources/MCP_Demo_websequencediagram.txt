note right of MCP Server: Checks intent

note right of MCP Server: Prepares API Call payload

MCP Server->MCP Client: returns weather context
note right of MCP Client: Builds enriched prompt
MCP Client->OpenAI: Sends enriched prompt
OpenAI->MCP Client: Returns a fluent answer
